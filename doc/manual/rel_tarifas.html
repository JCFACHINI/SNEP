Cadastros de Tarifas

Esta rotina exibe uma Lista de todas as Tarifas que estejam Cadastradas no Sistema.

É exibida sempre a tarifa mais recente de cada "condição" de tarifação cadastrada no sistema.

  •   Incluir Tarifas: Permite Cadastrar uma nova Tarifa.
  • Operadora, país, Cidade, Estado, Prefixo, DDD, Tarifas São informações sobre o Cadastro de cada Tarifa.
  • Ações: As ações são relativas a Tarifa listada na Linha correspondente:
    • Permite ALTERAR Valor de uma Tarifa.

    • Permite EXCLUIR uma Tarifa.
