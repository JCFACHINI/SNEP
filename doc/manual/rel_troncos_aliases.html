  •   Incluir Aliases de Troncos: Permite Cadastrar Aliases de Troncos ;

  • Ações: As ações são relativas ao Grupo de Captura listado na Linha correspondente:
    • Permite ALTERAR o Alias / Tronco.

    IMPORTANTE: Ao EXCLUIR um Alias de Tronco, os Cadastros de Troncos que tem correspondência com este cadastro ficarão órfãos, o que poderá ocasionar problemas no Sistema de Telefonia.
