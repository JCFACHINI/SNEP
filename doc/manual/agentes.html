Cadastro de Agentes

Esta rotina do sistema permite que Agentes sejam Cadastros/Configurados para as Filas de Atendimento.

Um Agente pode ser:
  • Estático: Quando ele permanece sempre ativo em uma ou mais fila de atendimento.
  • Dinâmico: Quando ele precisa efetuar ações de Login e Logoff para entrar e sair da Fila de Atendimento.

Os códigos do Agentes devem ser pré-definidos através das Configurações do Sistema. Para isso, acesso o Menu: Configurações >> Parâmetros e defina os códigos na opção "Range de Códigos para Agentes". A Figura a seguir mostra um ambiente com alguns códigos de agentes em uso e outros não.

Código: é o Código do Agente. A Lista de Códigos exibida, é baseada no Intervalo de Códigos configurado na Rotina Parâmetros (Menu: Configurações >> Parâmetros).

Nome: Define um Nome para o Agente (Ex: João da Silva)

Senha: Define uma senha para o Agente poder efetuar o LOGIN na Fila. A senha DEVE ser EXCLUSIVAMENTE NUMÉRICA.

Ações
  • Ativar / Desativar - determina se o Agente poderá ou não usar o sistema de telefonia.
  • - determina em qual(is) fila(s) o Agente será logado automaticamente pelo Asterisk. (v+: Filas do Agente)
