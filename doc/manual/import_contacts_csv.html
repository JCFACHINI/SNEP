Importação de Contatos via CSV

Para importar contatos basta selecionar um arquivo no formato CSV (Comma Separated Value). O arquivo deve ter os campos separados por virgulas.

Relacionamento dos Campos

Após fazer o upload do arquivo será mostrada uma tela onde é possível fazer o relacionamento dos campos do arquivo csv com as informações dos contatos. Basta selecionar quais colunas representam quais valoes nas caixas de seleção no cabeçalho da tabela.
