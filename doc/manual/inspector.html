Verificador do Sistema

Esta rotina permite um diagnóstico da instalação do SNEP, é possível verificar:

AGI : Verifica permissão dos arquivos de SNEP que interagem com o asterisk.

EXTENSIONS : Verifica se estão presentes no ambiente extensões do PHP necessárias para o bom funcionamento do SNEP.

LOGS : Verifica a existência e permissões dos arquivos de log SNEP.

PERMISSIONS : Verifica permissões dos arquivos do SNEP para o funcionamento do SNEP.

Na existência de problemas na sua instalação do SNEP, os mesmos são reportados nesta tela.
