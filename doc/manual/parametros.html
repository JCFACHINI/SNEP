Configurações - Parâmetros

Esta rotina permite que algumas configurações de usabilidade do SNEP sejam definidas ou ajustadas pelo usuário.

  • Nome da Empresa: Define o nome da Empresa onde o sistema está instaldo
  • Número de linhas ... : Define o número de linha que deverá ser exibida em cada página dos relatórios.
  • IP para Conexão (*) ... : Define um número IP ou Hostname do servidor onde o Asterisk está instalado.
  • Nome do Usuário (*) ... : Define o nome do usuário que será usado para conectar-se com o Asterisk.
  • Senha do usuário (*) ... : Define a senha do usuário que será usada para conectar-se com o Asterisk.
    • (*) A ligação do SNEP, através destes campos se dá com o Asterisk através do arquivo /etc/asterisk/manager.conf
  • Unidade de medida ... : Define o tipo de TEMPO (minutos ou segundos) que será usado nos relatórios.
  • Lista de Excessões ... : Define uma lista de ocorrencias a serem descartadas quando estas aparecerem no campo "dst" em um registro de chamada.
  • Aplicação para Sala de ... : Define que tipo de aplicação o Asterisk devera usar para gerenciar as Salas de Conferencias. Ao Alterar esta opção , acesse a rotina Cadastros >> Sala de Conferencias e clique no botão Gravar.
  • Modo de Debug: Ativa o debug para erros do php e aumenta o numero de mensagens na cli do asterisk feitas pelo agi do snep.

Configuração de gravação

  • Aplicação para Gravação das ... : Define que tipo de aplicação o Asterisk devera usar para efetuar as gravações das ligações.
  • Opções da aplicação de gravação : Define a extensão que foi usada pelo Asterisk para gravar chamadas no sistema.
  • Local dos arquivios ... :Define o Local onde estão os arquivos de gravação das chamadas.
  • Local do backup compactado ... : Define o Local onde estão os arquivos de backup ,compactados ,de gravação das chamadas.

Configurações de Ramais

  • Range de Ramais: Define limites para os numeros usados nos ramais.

Módulo de Gestão

  • Converter arquivos de som... : Define se um arquivo de som a ser usado no Asterisk deverá ser convertido para o formato GSM quando do seu Download para o Sistema. Este valor é apenas sugestivo, podendo ser alterado na rotina específica para importar arquivos de sons.
  • Range de Códigos para Agentes : Define o Intervalo (Inicio->Fim) de números que serão usados para nomear os Agentes do Sistema.

Filas

  • Tempo de atualização ... : Define o intervalor em SEGUNDOS que o sistema irá reler os dados do servidor Asterisk para exibir informações atualizadas do Status das Filas.
  • Número de Ligações ... :Define o número de ligaçõe que deverão estar em espera para que o sistema emite o alerta visual/sonoro.
  • Tempo Máximode espera ... : Tempo em SEGUNDOS que uma ligação em espera pode atingir para que o sistema emite o alerta visual/sonoro.

Troncos

  • Valor padrão para controle de qualidade: Valor padrão para o parâmetro qualify da configuração de troncos SIP.

Configurações de Discagem

  • Intervalo de repetição do alerta: Ao ativar controle de tempo define o intervalo em que o alerta sonoro será repetido ao atendente.
  • Arquivo de som para o alerta: Define o arquivo de som para o alerta de limite de tempo na ligação.

Módulo Painel

  • Aba do painel 1,2,3 :Define o nome que cada ABA do Painel - Flash Operator Panel. O Painel mostra todos as opções (ramais, troncos,agentes,...) configuradas no Asterisk e provê ao usuário, funcionalidades e ações sobre as chamadas , como: monitorar, encaminhar, transferir, atender, etc ...
