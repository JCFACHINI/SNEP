Contatos

Contatos são usados para facilitar algumas ações dentro da central SNEP. Podem ser usados para discagem abreviada usando códigos de atalho e para a manutenção de listas de telefones para uso com as regras de negócio.

Exclusão de Contatos em Lote

A exclusão de contatos em lote por enquanto se dá fazendo uma pesquisa com os filtros na lista de contatos e usando a opção "Apagar dados da seleção".

Cadastramento de Números:

Você facilmente pode adicionar números de contatos, desde que seu usuário tenha permissão para tal tarefa. Clique em Incluir Cadastros e preencha o formulário com as informações.

Acesso a funcionalidade:

*12 + [codigo]: Discar para número de telefone fixo.
*13 + [codigo]: Discar para número de telefone celular.
