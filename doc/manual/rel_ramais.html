Relação de Ramais

Esta rotina permite que sejam definidos os ramais que farão parte do sistema. Os Ramais definem os membros dos sistemas Asterisk e SNEP. Para o cadastro de ramais é necessário que se ajuste, em Configurações >> Parâmetros, os valores dos campos referentes a Tecnologias de Canais conforme for o caso, ou seja:
- Ramais SIP
- Ramais IAX2
- Ramais KHOMP

  •   Incluir Ramal: Permite Cadastrar um Novo Ramal ;

  • Incluir Ramais: Permite Cadastrar VARIOS Ramais;

  • Ramal, Nome completo, Canais, Contexto, Vinculos, Usa Voicemail São algumas informações sobre cada Ramal listado.
  • Ações: As ações são relativas ao Ramal listado na Linha correspondente:

    • Permite ALTERAR o cadastro de um Ramal ;

    • Permite EXCLUIR o cadastro de um Ramal;

    • Permite definir as PERMISSÕES de acesso ao Sistema de um Ramal;
