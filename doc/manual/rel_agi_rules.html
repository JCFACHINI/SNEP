Regras de Negócio

Esta rotina exibe uma Lista de todos as Regras de AGI que estejam Cadastrados no Sistema.

Os AGI são scripts ou programas que trabalham juntamente com Asterisk em "tempo real", ou seja, eles executam ações dependendo de determinadas condições.

  •   Incluir Regras de AGI : Permite Cadastrar novas Regras de AGI.

  •   Simulador : Permite simular uma ligação no Snep.

  • Código,Origem, Destino, Descrição São informações sobre o Cadastro de cada Regra de AGI.

  • Ações: As ações são relativas a cada Regra de AGI listada na Linha correspondente:
    • Permite ALTERAR dados de uma Regra de AGI.

    • Permite EXCLUIR uma Regra de AGI.
