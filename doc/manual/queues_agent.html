Filas do Agente

Esta rotina do sistema permite que um Agente seja Cadastrado em uma ou mais Filas de Atendimento.

Esta funcionalidade é particularmente útil para que o Agente, ao efetuar Login no Sistema de Telefonia, seja automaticamente colocado como disponível nas Filas a que ele pertence. O LOGIN em todas as Filas a que o Agente pertence é feito automaticamente pelo Sistema, assim como o LOGOFF.

Após selecionar a(s) Fila(s) desejada(s), mova-a(s) para a "Caixa: Membro das Filas" e então clique no botão "Gravar".
