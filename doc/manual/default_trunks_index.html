Relação de Troncos

Esta rotina permite que sejam definidos os Troncos que farão parte do sistema. Os Troncos definem as rotas de Entrada e saída das ligações do sistema Asterisk.

  • Incluir Tronco: Permite cadastrar um novo tronco.
Ações: As ações são relativas ao Tronco listado na Linha correspondente:
  • Permite ALTERAR o cadastro de um tronco.
  • Permite EXCLUIR o cadastro de um tronco.
