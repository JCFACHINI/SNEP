Atualizão de CNL - Cadastro Nacional de Localidades

Essa funcionalidade é responsável por atualizar os dados referentes a Operadora, Estado, cidade, DDD e prefixos de telefones do Brasil.

Para atualizar CNL:

Você facilmente pode atualizar os dados da CNL, desde que tenha o arquivo fornecido pelo departamento comercial.
Para tal tarefa, clique em Enviar arquivo e indique o arquivo da CNL. Depois é só clicar no botão Salvar.
