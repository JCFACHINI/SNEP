Salas de Conferência

Esta rotina do sistema permite que Salas de Conferências seja Ativadas e/ou Desativadas no Sistema de Telefonia.

  • Código: A Lista de Códigos (Sala de Conferência) exibida, é baseada no Intervalo de Códigos configurado na Rotina Parâmetros (Configurações >> Parâmetros).
  • Usa Autenticação: Define se para usar a Sala de Conferência será necessário o uso de Uma Senha.
  • Senha de Autenticação: Define uma Senha para a Sala de Conferência - a senha deve ser EXCLUSIVAMENTE NUMÉRICA.
  • Centro de Custos:- determina para qual Centro de Custos as chamadas de Conferências serão atribuidas.
  • Ações:
    • Ativar / Desativar - determina se a Sala de Conferência está Ativa ou Não
