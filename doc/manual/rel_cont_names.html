Agenda de Contatos

Contatos é uma simples agenda de endereço e telefones fixo e celular utilizada principálmente para discagem abreviada.

A funcionalidade de discagem abreviada permite que o usuário use um código do SNEP + código do Cadastro do Contato para efetuar a chamada diretamente, sem a necessidade de discar o número do telefone fixo ou celular. Cadastramento de Números:

Você facilmente pode adicionar números de contatos, desde que seu usuário tenha permissão para tal tarefa. Clique em Incluir Cadastros e preencha o formulário com as informações.

Acesso a funcionalidade:

*12 + [codigo]: Discar para número de telefone fixo.
*13 + [codigo]: Discar para número de telefone celular.
