Regras de Negócio - Configurações Padrão

Permite a interação com configurações padrões de ações das regras de negócio. Cada ação pode determinar um conjunto de configurações padrões para serem usadas por elas, estas são processadas em um formulário nessa tela para que possam ser controladas a partir da interface.

Para documentação detalhada das configurações de ações de módulos consulte a documentação do módulo.
