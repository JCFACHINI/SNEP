Cadastros de Grupos de Captura

Esta rotina exibe uma Lista de todos os Grupos de Captura que estejam Cadastrados no Sistema.

Os Grupos de Captura fazem parte do Cadastro de um Ramal e ajudam a determinar o Ramais que poderão capturar ligações entre si.

  • Incluir Grupos de Captura: Permite Cadastrar novos Grupos de Captura.
Ações: As ações são relativas ao Grupo de Captura listado na Linha correspondente:
  • Permite ALTERAR dados do Grupo de Captura.
  • Permite EXCLUIR o Grupo de Captura ;

IMPORTANTE: Ao EXCLUIR um Grupo de Captura, os Ramais que tem correspondência com este cadastro ficarão órfãos, o que poderá ocasionar probblemas no Sistema de Telefonia.
