Logs do Sistema

Esta rotina permite uma pesquisa nos Logs do Snep. Toda ação realizada pelo Snep é registrada num arquivo. Esta ação pode ser uma ligação de um ramal para determinado número, uma transferência de chamada entre outros. Esta rotina serve para auxilia-lo a detectar possíveis problemas no funcionamento da central telefônica.

É possível verificar os logs de duas formas:
  • Pesquisa nos Logs : É uma pesquisa mais detalhada, procurando ações que foram realizadas apartir de determinada origem ou destino, ou intervalo de dias.
    • Período: Especifica um intervalo de tempo onde pretende-se pesquisar determinada ocorrência.
    • Status do Log: Cada informação registrada nos logs possue um grau de importância, é possível especificar tal relevância através dos Status.
    • Origem / Destino : Possibilita que você especifique detarminada origem ou destino, como por exemplo, um ramal ou um número de telefone.
  • Em tempo real : Este tipo de pesquisa mostra os logs em tempo real, cada informação registrada é mostradas na tela. As informações são atualizadas num intervalo de 1 (um) segundo.
