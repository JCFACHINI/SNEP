Cadastros de Filas

Esta rotina exibe uma Lista de todos as Filas de Atendimento Cadastradas no Sistema. As Filas são uma das funcionalidades que o SNEP tem para a implementação de um call center básico.

Filas permitem que vários ramais façam parte de um contexto em comum para que chamadas de entrada e/ou saída possam ser direcionadas para este.

  • Incluir Filas: Permite Cadastrar novas Filas de Atendimento.
Ações: As ações são relativas ao Centro de Custos listado na Linha correspondente:
  • Permite ALTERAR o cadastro da Fila.
  • Permite EXCLUIR o cadastro da Fila.
  • Permite DEFINIR os membros estáticos da Fila.
Obs: Ao cadastrar Filas através desta rotina, não esqueça de definir em Configurações >> Parâmetros que o Sistema deve obter a Lista de Filas a Partir do Banco de Dados ao invés deo Arquivo texto.
