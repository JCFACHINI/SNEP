Status dos Links

Esta rotina do sistema mostra informações sobre o Status de ERROS de Links de Placas Khomp. Se nenhuma placa Khomp for identificada pelo Sistema, uma mensagem de AVISO será exibida.

O botão [Zerar Contadores] , ao ser clicado, irá "reiniciar" os contadores de erros. Isto pode ser útil para uma efetivo monitoramento de de erros, já que o hardware não informa desde quando os contadores estã monitorando os links.
