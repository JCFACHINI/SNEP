Arquivos de Sons

Esta rotina permite ao usuário cadastrar e manipular arquivos de Sons do Asterisk, como URA's, mensagens de Boas vindas, Horário de funcionamento da empresa, etc.

Esta rotina apenas mantém um cadastro dos arquivos de som e permite que novos arquivos sejam inseridos ou trocados no diretório padrão do Asterisk. Esta rotina NÃO altera configurações do Asterisk, portanto os nomes de arquivos existentes NUNCA são trocados - estes somente são sustituídos.

  •   Cadastrar Arquivos de Som: Permite Cadastrar e/ou incluir um Arquivo de Som do Asterisk.
  • Nome, Descrição, Tipo, Atualizado ...: São informações sobre o Cadastro de cada Arquivo de Som.
  • Ouvir: As ações são relativas a cada Arquivo de Som listado na Linha correspondente:
    • Permite OUVIR o Arquivo de Som de backup ou o que está em uso (atual) ;

  • Ações: As ações são relativas a cada Arquivo de Som listado na Linha correspondente:
    • Permite ALTERAR dados ou trocar o Arquivo de Som.

    • Permite RESTAURAR um backup.

    • Permite EXCLUIR o cadastro de um Arquivo Som - O Arquivo não será excluido.
