Status das Filas

Para que a lista de Filas Disponíveis esteja visível é necessário que as mesmas estejam definidas como "Autorizadas" no cadastro do usuário/ramal em uso (veja o Menu: Cadastros >> Ramais).

Filas: Selecione a(s) Fila(s) que deseja monitorar selecionando­a(s) e movendo­a(s) para a "Caixa Selecionadas" usando a seta para direita.

Ver Status Físico dos Ramais: Selecionando "Sim", irá exibir, quando disponível, informações sobre a estação de trabalho onde o Ramal está conectado fisicamente. Esta opção é particularmente útil quando se usa Softphones (Software que simula uma telefone - veja capítulo específico sobre isso neste Manual) . Ao clicar no botão "Visualizar", uma tela com a Lista de todos os membros de cada Fila será exibida, assim como informações da Fila e de cada membro da fila:

Esta rotina permite que seja monitorado o Status de cada membro da(s) Fila(s) selecionada. Para que a lista de Filas Disponíveis esteja visível é necessário que estas estejam cadastradas e a origem do cadastros (arquivo texto ou banco de dados) esteja definido em: Configurações >> Parâmetros.

Após selecionar a(s) Fila(s) desejada(s), mova-a(s) para a "Caixa Selecionadas" e então clique no botão "Visualizar". Um tela com a Lista de todos os Membros de cada Fila será exibida, assim como algumas informações de cada membro da fila, como:

  • Status
  • Hora do Login
  • Tempo decorrido desde que a última ligação foi atendida
  • Quantas ligações foram atendidas
  • Número do Telefone com quem ele está falando
  • etc...

ALARMES

As Filas possuem um sistema de alerta que dispara 3 eventos distintos e configuráveis:

  • Alerta Sonoro
  • Alerta Visual
  • Alerta por E-mail

As condições de disparo destes Alertas podem ser configuradas de forma genérica, ou seja, para todas as filas, em Configurações >> Parâmetros, ou então , individualmente em Cadastros >> Filas
