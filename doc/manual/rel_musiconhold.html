Músicas em Espera

Esta rotina permite ao usuário manter um Cadastro de Arquivos para serem usados como "Músicas de Espera" pelo Asterisk.

As músicas em espera são agrupadas em "SEÇÕES" e por padrão, o SNEP cria três seções na sua instalação, além da seção "Default" do ASterisk. Os arquivos de som serão colocados nestas seções conforme for o casoe as seções serão usadas de acordo com o Plano de Discagem definido.

  •   Incluir Seções: Permite Cadastrar ou Alterar uma Seção.
  • Seção, Descrição, Modo, Diretório : São informações sobre o Cadastro de cada Seção.
  • Ações: As ações são relativas a cada Seção listada na Linha correspondente:
    • Permite ALTERAR dados da seção.

    • Permite EXCLUIR uma seção. Neste caso, todos os arquivos e o diretório da seção serão excluídos. A seção DEFAULT não pode ser excluída.

    • Permite CADASTRAR Arquivos de Som para a seção.
